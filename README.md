## 🚀 Как разместить сайт в интернете

### Вариант 1: GitHub Pages (Бесплатно, 5 минут) ⭐

**Самый простой способ!**

1. Создайте аккаунт на [GitHub.com](https://github.com)
2. Создайте новый репозиторий с названием `havana-club`
3. Загрузите все файлы из папки `website` в репозиторий
4. Перейдите в Settings → Pages
5. В разделе "Source" выберите "main" ветку
6. Нажмите Save

Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/havana-club`

**Видео-инструкция:** [YouTube - GitHub Pages Tutorial](https://www.youtube.com/results?search_query=github+pages+tutorial)

---

### Вариант 2: Netlify (Бесплатно, автоматический) 🚀

**Современный и быстрый!**

1. Зарегистрируйтесь на [Netlify.com](https://netlify.com)
2. Нажмите "Add new site" → "Deploy manually"
3. Перетащите папку `website` в окно браузера
4. Готово! Сайт будет опубликован за 30 секунд

**Преимущества:**
- Бесплатный SSL сертификат (https://)
- Быстрый CDN
- Можно подключить свой домен
- Автоматические обновления

- Telegram (сейчас: @Kaydunov_Denis)
- Адрес (сейчас: г. Гомель, ул. Карповича, 45)

### Изменить цены:

Откройте `index.html` и найдите секцию `<!-- Pricing Section -->`, измените цифры в:
```html
<span class="price-amount">5</span>
```

### Изменить расписание:

Откройте `index.html` и найдите секцию `<!-- Schedule Section -->`, измените время и названия занятий.

### Изменить цвета:

Откройте `styles.css` и найдите раздел `:root`, измените:
```css
--primary-color: #E63946;      /* Красный */
--secondary-color: #F77F00;    /* Оранжевый */
```
